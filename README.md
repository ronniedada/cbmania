cbmania
=======

web-based couchbase interactive shell  - trycouchbase.com

Prerequisites
-------------

* node.js
* couchnode

Dependencies
------------

    npm install express

    npm install nconf

    npm install stylus

    npm install jade

    npm install now

Build & Start
------------
   npm install
   node app.js

point web browser at http://127.0.0.1:5206

Configuration
-------------
    config.json

Supported Commands
------------------

Telnet:

    set [key] [val]
    get [key]
    del [key]
